# SEO-гид для блога 2shezlonga

## Структура URL
/blog                          — листинг всех статей
/blog/[slug]                   — страница статьи
/sitemap.xml                   — автогенерация из sitemap.ts
/robots.txt                    — автогенерация из robots.ts

## Метатеги на каждой статье
- title: уникальный, 50-60 символов, ключевое слово в начале
- description: 140-160 символов, призыв к действию
- canonical: всегда прописывай абсолютный URL
- og:type = "article" для статей, "website" для листинга
- og:image: 1200×630px, JPG, до 200 КБ

## JSON-LD схемы
- BlogPosting — на каждой статье
- FAQPage — если есть FAQ-блок (попадает в Google People Also Ask)
- BreadcrumbList — на каждой статье
- WebSite + SearchAction — один раз в корневом layout

## FAQ-секция
- Используй тег <details>/<summary> — Google и Яндекс понимают
- Минимум 3 вопроса на статью
- Вопросы формулируй как реальные поисковые запросы
- Ответ — 40-60 слов, конкретный факт
- Попадает в "Люди также спрашивают" и в AI-сниппеты (ChatGPT, Алиса)

## Внутренняя перелинковка
- Из каждой статьи — 2-4 ссылки на другие статьи блога (relatedSlugs)
- Каждая статья → CTA-блок → страница туров по направлению
- Листинг блога → ссылки на статьи с rich anchor text
- Футер → популярные статьи + направления туров
- Хлебные крошки на каждой странице

## Для Яндекса
- Добавь сайт в Яндекс.Вебмастер
- Укажи region в настройках
- Заполни Турбо-страницы (опционально)
- robots.txt: укажи Host: https://2shezlonga.ru

## Для AI-ассистентов (ChatGPT Browse, Алиса, Perplexity)
- Чёткие заголовки H2/H3 с вопросительными формулировками
- Короткие абзацы (3-5 строк)
- Списки вместо сплошного текста
- Числовые данные и даты — роботы цитируют их охотнее
- FAQ-блок с schema FAQPage

## Checklist перед публикацией статьи
[ ] Уникальный title + description
[ ] canonical URL
[ ] og:image 1200×630
[ ] JSON-LD: BlogPosting + FAQPage + Breadcrumb
[ ] Минимум 3 FAQ-вопроса
[ ] 2+ ссылки на похожие статьи (relatedSlugs)
[ ] CTA-блок со ссылкой на туры
[ ] alt у всех картинок
[ ] slug только строчными буквами через дефис
[ ] Статья добавлена в blogPosts в lib/blog.ts
[ ] sitemap.ts обновится автоматически
